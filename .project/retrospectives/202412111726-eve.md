## 🔧 Technical Details

### CI/CD Authentication Fix
```yaml
# Before (didn't work)
permissions:
  id-token: write
  contents: read

# After (works with NPM_TOKEN)
permissions:
  id-token: write
  contents: read
  packages: read

- name: Configure npm registry auth
  run: |
    echo "//npm.pkg.github.com/:_authToken=${{ secrets.NPM_TOKEN }}" > ~/.npmrc
    echo "@askend-lab:registry=https://npm.pkg.github.com" >> ~/.npmrc
```

### Jest Configuration Fix
```javascript
// Before (didn't work with pnpm)
testPathIgnorePatterns: ['/node_modules/']

// After (works)
testMatch: ['<rootDir>/test/**/*.test.js', '<rootDir>/packages/*/test/**/*.test.js']
```

### Terraform Local Testing
```bash
# Format check
terraform fmt -check

# Validation
terraform validate

# Plan (read-only, safe with -lock=false)
terraform plan -var="env=dev" -lock=false
```
