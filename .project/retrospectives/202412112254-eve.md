## 🎯 Key Learnings

1. **Terraform workflow triggers require path matches** - `.github/workflows/terraform.yml` on: push: paths: `infra/**` means only infra/ changes trigger it
2. **Separate state files per environment is critical** - Shared state causes resource conflicts
3. **AWS CloudFront needs account verification** - New AWS accounts can't create CloudFront distributions without support ticket
4. **Terraform import with || true is safe** - Allows graceful handling of missing resources
5. **S3 static hosting is viable fallback** - Can serve HTTP sites directly without CloudFront
6. **Provider configs must exist for state cleanup** - Can't remove provider if resources in state use it
7. **React hooks make great live UIs** - useState + useEffect for live clock works perfectly

## 🔧 Technical Details

### Terraform State Separation
```yaml
# Before (broken - shared state)
backend "s3" {
  key = "hak/terraform.tfstate"
}

# After (fixed - per-environment state)
backend "s3" {
  # Key set via -backend-config
}

# In workflow
terraform init -backend-config="key=hak/${{ matrix.environment }}/terraform.tfstate"
```

### Import Step for Existing Resources
```yaml
- name: Import existing resources (if any)
  run: |
    terraform import -var="env=${{ matrix.environment }}" aws_s3_bucket.website hak-${{ matrix.environment }}-website || true
    terraform import -var="env=${{ matrix.environment }}" aws_s3_bucket_website_configuration.website hak-${{ matrix.environment }}-website || true
    # ... more imports
  continue-on-error: true
```

### S3 Static Hosting Configuration
```hcl
# Route53 pointing to S3 website endpoint (not CloudFront)
resource "aws_route53_record" "website" {
  alias {
    name       = aws_s3_bucket_website_configuration.website.website_domain
    zone_id    = aws_s3_bucket.website.hosted_zone_id
  }
}
```

### Modern Landing Page Features
```tsx
// Live clock with React hooks
const [time, setTime] = useState(new Date())

useEffect(() => {
  const timer = setInterval(() => setTime(new Date()), 1000)
  return () => clearInterval(timer)
}, [])

// Display
{time.toLocaleTimeString()}
```
