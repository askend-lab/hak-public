# HAK

Estonian language learning platform — serverless LMS where teachers create interactive lessons and students complete them.

Built for AI-assisted development using [DevBox](devbox.yaml) — a toolkit that enforces TDD, coverage thresholds, and code quality through git hooks.

## Quick Start

```bash
pnpm install
pnpm run dx    # Full check: tests + hooks
pnpm start     # Run frontend + backend locally
```
