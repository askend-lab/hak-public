# EN 301 549 — EU Public Sector Accessibility Checklist

> https://www.etsi.org/deliver/etsi_en/301500_301599/301549/
> **Legally mandatory** for Estonian public sector web applications.
> Format: [ ] **check** = verification exists · [ ] **done** = requirement satisfied

## Chapter 9: Web (maps to WCAG 2.1 AA)
- [ ] check · [ ] done — All WCAG 2.1 AA criteria met (see [wcag.md](./wcag.md)) (`axe-core`)
- [ ] check · [ ] done — Conformance level documented in Accessibility Statement (`manual review`)

## Chapter 11: Software
- [ ] check · [ ] done — Platform a11y features not disrupted (`Playwright + screen reader`)
- [ ] check · [ ] done — User preferences for font/colors respected (`Playwright viewport tests`)
- [ ] check · [ ] done — No interference with assistive tech (`manual screen reader test`)

## Chapter 12: Documentation and Support
- [ ] check · [ ] done — Documentation is accessible (HTML, not PDF-only) (`manual review`)
- [ ] check · [ ] done — A11y features documented for users (`manual review`)
- [ ] check · [ ] done — Support channels accessible (`manual review`)

## Accessibility Statement (required by EU directive)
- [ ] check · [ ] done — Published on the website in accessible format (`Playwright test`)
- [ ] check · [ ] done — Contains: conformance status, non-accessible content (`manual review`)
- [ ] check · [ ] done — Contains: date of assessment, method (`manual review`)
- [ ] check · [ ] done — Updated annually (`calendar reminder`)
- [ ] check · [ ] done — Link to enforcement body (Estonian TTJA) (`manual review`)

## Testing Requirements
- [ ] check · [ ] done — Automated testing with axe-core in CI (`run-tests` + Playwright)
- [ ] check · [ ] done — Manual keyboard-only navigation test (`manual checklist`)
- [ ] check · [ ] done — Screen reader test (NVDA or VoiceOver) (`manual checklist`)
- [ ] check · [ ] done — 200% zoom test (`Playwright viewport test`)
- [ ] check · [ ] done — 320px viewport test (`Playwright viewport test`)
