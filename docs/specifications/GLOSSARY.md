# Glossary - EKI Pronunciation Learning Platform

## Core Terms

**Vabamorf** - Estonian morphological analyzer that adds stress markers to text  
**Merlin TTS** - Text-to-speech synthesis engine for Estonian language  
**Stress markers** - Phonetic annotations indicating syllable stress in Estonian words  
**Morphological analysis** - Process of analyzing word structure and adding linguistic annotations

## User Interface Terms

**Kõnesüntees** - Speech synthesis (Estonian)  
**Kõnevoor** - Speech queue/playlist (Estonian)  
**Ülesanded** - Tasks (Estonian)  
**Tagasiside** - Feedback (Estonian)  
**Lausungid** - Utterances/sentences (Estonian)

## Technical Terms

**eID** - Estonian digital identity system  
**Isikukood** - Estonian personal identification code (11 digits)  
**Smart-ID** - Estonian digital authentication method  
**Mobiil-ID** - Estonian mobile authentication method  
**ID-kaart** - Estonian national ID card

## Feature Terms

**Playlist** - Collection of synthesized audio items for sequential playback  
**Task** - Structured collection of utterances for learning purposes  
**Phonetic display** - Visual representation of pronunciation with stress markers  
**Share link** - Public URL for accessing shared tasks without authentication
